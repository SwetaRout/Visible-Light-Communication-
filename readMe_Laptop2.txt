Laptop 2

Function :
1)  For mode 1, display characters typed in laptop 1 character by character
2)  For mode 2, receive a file from laptop 1 in a zipped format received_file.zip. User just have o extract it to get the transferred file.

How to run :

1) Copy the file laptop2.py and go to its location
2) Open command window(for windows) or linux terminal and type the follwing command
	python laptop2.py

Note:

You can change the port number and host
